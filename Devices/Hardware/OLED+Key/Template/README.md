# README

## 1. 程序说明:

功能

**基础功能:**

* `Systick` : 1ms一次中断回调

* OLED : 展示数据
* 按键(Key) 两个,非阻塞



## 2. 引脚及功能说明

### 2-1 OLED

| 引脚 | 标签         |
| ---- | ------------ |
| PB8  | OLED_SCL_Pin |
| PB9  | OLED_SDA_Pin |

### 2-2 KEY

| 引脚 | 标签     |
| ---- | -------- |
| PB12 | KEY1_Pin |
| PB13 | KEY2_Pin |
